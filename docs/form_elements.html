Fieldset and Legend

Legend

Fieldsets and legends should primarily be used for semantic-separation, as they are difficult to style.

<form>
    <fieldset>
        <legend>Legend</legend>
        <p>Fieldsets and legends should primarily be used for semantic-separation, as they are difficult to style.</p>
    </fieldset>
</form>

Search Field

This will expand to fill the width of its parent container. Sizes are set in EMs so it can inherit it's parent element's size properties.

    <div class="search-container">

</form>
