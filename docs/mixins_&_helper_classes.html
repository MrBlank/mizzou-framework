Clearfix

The .clearfix() mixin is way to apply a clear fix to an element without extra markup.

Box

The .box() mixin is good for adding empasis to an element.

A box with some added style options.

<div class="box-example">
    <p>A box with some added style options.</p>
</div>

Here are the variables that can be used with their defaults:

@background: @color-box-bg          // Use 'none' to remove
@padding: 1                         // This is a REM unit 
@border-size: 0                     // Change this to add a border
@border-color: @color-box-border 
@border-style: solid

.box-example {
    .box(@gold-lightest, 1.5, 1px, @gold);
}

Center Block

The .center-block() mixin is a quick way to center layouts.

A Centered, block-level element.

<div class="center-me">
    <p>A Centered, block-level element.</p>
</div>

Here are the variables that can be used:

@block-max-width  
@block-min-width

.center-me {
    .center-block(@block-max-width: 300px, @block-min-width: 150px);
    .box();
    width: 50%;
}

Image Replace

.img-replace() is an accessibility trick to allow use of image sprites for text. Size the element to your image and this indents the text off-screen.

Mizzou Logo 

<p class="img-replace-example">
    <a href="http://missouri.edu">Mizzou Logo</a>
</p>
.img-replace-example {
    .img-replace();
    width: 64px;
    height: 72px;
    background-image: url("@{img-path}/mizzou-logo-64.svg");
}

Last Child

The .last-child() mixin overrides the bottom margin of the last child element to 0. This helps keep padding on the parent element consistant. Used in the .box() mixin.

Type Helpers

Hanging Quotes

The .quo and .dquo classes are for single quotes and double quotes, respectively. They push the element to the left a little to cause the punctuation to hang into the margins.

‘Let these columns stand. Let them stand a thousand years.’

“Let these columns stand. Let them stand a thousand years. Crown them with an arch, memorial to the men who in their magnificent presence learned what life and duty are, and how to live the one and do the other. They will be to all the rallying point of future devotion and service to the university.”

<h1><span class="quo">&#8216;</span>Let these columns stand. Let them stand a thousand years.&#8217;</h1>
<p><span class="dquo">&#8220;</span>Let these columns stand. Let them stand a thousand years. Crown them with an arch, memorial to the men who in their magnificent presence learned what life and duty are, and how to live the one and do the other. They will be to all the rallying point of future devotion and service to the university.&#8221;</p>

Small Caps

The .caps class shrinks text a little and transforms text to uppercase.

The quarterly magazine of MAA is called MIZZOU and provides news and information about the University of Missouri

<p>The quarterly magazine of <span class="caps">MAA</span> is called <span class="caps">MIZZOU</span> and provides news and information about the University of Missouri</p>

Dropcaps

The .dropcap class works on an inline element, usually and span tag. Add the class .hang when it starts with punctuation to hang it in the margins.

Skip to Content

The "Skip-To" link is an accessibility technique to allow users to skip directly to navigation or content. .offset should be the first element inside the <body> tag. You can see what it looks like by tabbing through the document. More skip links can be added as necessary.

<div class="offset">
    <a href="#nav" tabindex="1">Skip to navigation</a>
    <a href="#content" tabindex="2">Skip to content</a>
</div>
