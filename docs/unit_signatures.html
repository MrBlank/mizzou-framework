Unit Signature Mixin

The .sig() mixin sets the base styles for all signatures.

The .sig-options() mixin has a set of variables that can be passed to it to define size, style and printing options. This is also useful in resetting options for media queries. The .sig() mixin does not need to be reset.

These are the options:

@sig-size:          "64" // Default 
@sig-size:          "48"
@sig-size:          "32"

@logo-style:        "none" // Default 
@logo-style:        "stroked"

@wordmark-color:    "black" // Default 
@wordmark-color:    "white"

@wordmark-indent:   0 // Default
@wordmark-indent:   2px // Use a unit value to add left margin to wordmark

@print:             "no" // Default
@print:             "yes"

@center:            "no" // Default
@center:            "yes"

@unit-margin:       0 // Default
@unit-margin:       2px // Use a unit value to add bottom margin to unit name

These unit variables set the options for the custom unit name image. Every site will have a unique unit name. Images for these are not part of the framework.

All three variables are required.

@unit-width:        100px // Use px units
@unit-height:       25px
@unit-image:        "image-file-name.svg" // Place image in images folder

Unit Signature IE8 Fallback

There are PNG fallback mixins for browsers that do not support SVG images. When needed, use the .sig-options-ie() mixin in your IE specific stylesheet to override the SVG image with a PNG version.

.sig-options-ie() must be used in conjunction with the .sig() and .sig-options() mixins and uses some of the same variables set in .sig-options().

These must be set again if used in .sig-options().

@sig-size 
@logo-style 
@wordmark-color 
@print
@unit-img // Required

Mizzou Logo

Ofﬁce of Programs

University of Missouri

<div class="ie">
    <p class="logo"><a href="http://missouri.edu">Mizzou Logo</a></p>
    <h1 class="unit"><a href="http://missouri.edu">Of&#64257;ce of Programs</a></h1>
    <h2 class="wordmark"><a href="http://missouri.edu">University of Missouri</a></h2>
</div>
.ie {
    .sig(); 
    .sig-options(@sig-size: "64", @print: "yes", @unit-width: 404px, @unit-height: 57px, @unit-img: "unit-name_64.svg");
}

// Override for IE8 in your IE8 stylesheet
.ie {
    .sig-options-ie(@sig-size: "64", @print: "yes", @unit-img: "unit-name_64.png"); 
}
