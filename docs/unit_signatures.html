Unit Signature Mixin

The .mu-unit-sig() mixin has a set of variables that can be passed to it to define size, style and printing options. This is also useful in resetting options for media queries. Below are the mixin options.

@logo-size:          "64" 
@logo-size:          "48" // Default
@logo-size:          "32"

@logo-style:        "stroked" // Default 
@logo-style:        "none"

@wordmark-color:    "black" // Default 
@wordmark-color:    "white"

@wordmark-indent:   0 // Default
@wordmark-indent:   2px // Use a unit value to add left margin to wordmark

@print:             "no" // Default
@print:             "yes"

@center:            "no" // Default
@center:            "yes"

@unit-margin:       0 // Default
@unit-margin:       2px // Use a unit value to add bottom margin to unit name

The unit variables below set the options for the custom unit name image. Every site will have a unique unit name. Images for these are not part of the framework. All variables Except @unit-img-print are required.

@unit-width:        100px // Use px units
@unit-height:       25px
@unit-image:        "image-file-name.svg" // Place image in images folder
@unit-img-print:    "image-file-name-print.svg" // Defaults to @unit-image

Unit Name Image Creation

University units may not create new logos. Unit signatures have a carefully defined relationship among the logo, the unit and division names, and the university wordmark. For this reason, do not attempt to re-create any part of a unit signature. To obtain unit name graphics, contact the Division of Marketing & Communications at identity@missouri.edu. See section 1.4 of the Graphic Identity Standards at identity.missouri.edu.

When designing for a small phone screen, the max-width the unit name can be is 245px wide.

Unit Signature IE8 Fallback

There is a PNG fallback mixin for browsers that do not support SVG images. When needed, use the .mu-unit-sig-ie() mixin in your IE specific stylesheet to override the SVG image with a PNG version.

.mu-unit-sig-ie() must be used in conjunction with the .mu-unit-sig(). The variables below must be set again if used in .mu-unit-sig().

@logo-size 
@logo-style 
@wordmark-color 
@print
@unit-img // Required
@unit-img-print // Defaults to @unit-img
