Print

Print Reset

The _print.less stylesheet does the following to all elements on the page to help with printer compatibility:

  • Makes text black
  • Makes borders black
  • Removes backgrand images
  • Makes background colors white

It also overrides the base font size and sets it to 11px on the html tag.

Individual sites will need a separate set of rules to hide unnecessary block level elements and make fine-tuned adjustments.
